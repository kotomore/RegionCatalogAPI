# Region Catalog API

## Based
#### Java + Spring Boot + H2 + MyBatis


## Swagger
http://localhost:8080/swagger-ui/index.html#/


## Install && launch

#### Requirements: Java 17

```
git clone https://github.com/kotomore/RegionCatalogAPI.git
cd RegionCatalogAPI
mvn package
java -jar target/RegionCatalogAPI-0.0.1-SNAPSHOT.jar
```
